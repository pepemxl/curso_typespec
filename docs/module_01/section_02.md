# Configuración del Entorno

Consiste normalmente de los siguientes dos pasos:

1. Instalación de Node.js y npm
2. Instalación del compilador TypeSpec

## Requerimientos


- `node.js 20 LTS`
- `npm 7+`


## Instalando typespec

`npm install -g @typespec/compiler`
